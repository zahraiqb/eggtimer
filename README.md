# Egg Timer App

## Overview

The Egg Timer App is a simple React Native application built using Expo that helps users cook different types of eggs perfectly. Users can select the type of egg they want, and a timer will start to guide them through the cooking process.

## Technologies Used
- React Native
- Expo
- JavaScript
- React Navigation

## Running the App

1. Install dependencies:
npm install

Start the app:
npx expo start

Open using:
Expo Go app on your phone, or iOS/Android emulator

The graphics used in the UI are also drawn by me (which is why all the eggs look like distant cousins)

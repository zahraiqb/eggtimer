# Egg Timer App

## Overview

A React Native application built using Expo that helps users cook different types of eggs perfectly. Users can select the type of egg they want, and a timer will start to signify how long the egg should be cooked. 

## Technologies Used
- React Native
- Expo
- JavaScript
- React Navigation

## Project Structure
- App.js – Navigation setup
- Screens/ – Contains code for all app screens
- assets/ – Images/graphics used in the app
